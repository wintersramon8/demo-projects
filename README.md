# Demo Projects

This repository contains four demo projects showcasing React Native mobile apps and their Next.js web counterparts.

## Projects

### Mobile Apps (React Native + Expo)

- **[RSVP Events Demo](./rsvp-events-demo/)** - A mobile app for discovering and managing events with RSVP functionality
- **[Social Platform Demo](./social-platform-demo/)** - A social networking app with feed, profiles, posts, comments, and follow features

### Web Apps (Next.js)

- **[RSVP Events Web](./rsvp-events-web/)** - The web version of the RSVP Events app
- **[Social Platform Web](./social-platform-web/)** - The web version of the Social Platform app

## Quick Start

Each project has its own README with setup instructions. Navigate to any project folder to get started.

All projects require:
- Firebase configuration (for authentication and database)
- Cloudinary configuration (for image uploads)

See each project's README for detailed environment variable setup.
